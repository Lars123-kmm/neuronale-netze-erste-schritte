# NeuronaleNetze_Erste_Schritte
Dieses Projekt orientiert sich inhaltlich am Buch
**„Neuronale Netze selbst programmieren – Ein verständlicher Einstieg mit Python“
von Tariq Rashid, übersetzt von Frank Langenau.
Das Buch dient mir als Grundlage, um die Funktionsweise neuronaler Netze Schritt für Schritt nachzuvollziehen und die Konzepte selbst in Python umzusetzen.
